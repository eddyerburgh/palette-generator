# palette-generator

> A Vue.js project

See it live [here](http://eddyerburgh.github.io/palette-generator/)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run all tests
npm test
```
## Dependencies

- Java > 8 for WebDriver tests
